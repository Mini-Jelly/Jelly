✅自定义首页轮播图(Swiper)，广告大图(Banner)

✅自定义头部代码，底部代码

✅支持文章缩略图设置，文章个性化标徽设置

✅支持选择加载方式为瀑布流🌊或分页器

✅支持文章二维码打赏、代码高亮

🔥支持公共资源选择CDN加载

## 😮如何使用

蓝奏云下载压缩包，解压到你的主题theme目录下即可，或者点击右边的发行版下载

## 🎮二次开发

1. 克隆本项目
```
git clone https://gitee.com/yangxingrui/jelly.git
```

2. 在终端加载项目

```
npm install
```

3. 开始编辑代码

推荐使用vscode作为编码器
